# admin-lte-world

基于 Vue.js 的 Star 收藏系统开发, 项目需要 nodejs 环境，基于 AdminLTE 后端模版。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
